Installation

Queryon is a modern, lightweight, and modular CSS framework designed to help developers build beautiful, responsive websites with ease. It provides a comprehensive suite of utility classes, a responsive grid system, and pre-built components.

CDN

Include the following link in your HTML file:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/SHADOWMONARCH-ARCH/Queryon-css@refs/heads/main/queryon-css.min.css">

Download

Download the CSS file and include it in your project:

<link rel="stylesheet" href="path/to/queryon.css">

Quick Start

Create your first webpage with Queryon using our starter template:

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My Queryon Project</title>
    <link rel="stylesheet" href="queryon.css">
  </head>
  <body>
    <div class="q-container">
      <h1 class="q-text-primary">Hello, Queryon!</h1>
      <p class="q-text-secondary">Welcome to your first Queryon project.</p>
    </div>
  </body>
</html>

Grid System

Queryon provides a responsive 12-column grid system built with flexbox. The grid system helps you create complex layouts that adapt to different screen sizes.

Basic Grid Structure

Use the .row class as a container for columns, and .col-* classes to define column widths.

<div class="row">
    <div class="col-6">50% width</div>
    <div class="col-6">50% width</div>
</div>

<div class="row">
    <div class="col-4">33.33% width</div>
    <div class="col-4">33.33% width</div>
    <div class="col-4">33.33% width</div>
</div>

Responsive Grid

Use responsive classes to change column widths at different breakpoints:

  • .col-sm-* - Small devices (576px)
  • .col-md-* - Medium devices (768px)
  • .col-lg-* - Large devices (992px)
  • .col-xl-* - Extra large devices (1200px)
<div class="row">
    <div class="col-12 col-md-6 col-lg-4">
        Full width on mobile, half on tablet, third on desktop
    </div>
</div>

Column Offsets

Use .offset-* classes to offset columns:

<div class="row">
    <div class="col-6 offset-3">Centered column with offset</div>
</div>

Auto-width Columns

Use .col-auto for columns that automatically size based on content:

<div class="row">
    <div class="col-auto">Auto width</div>
    <div class="col">Takes remaining space</div>
</div>

Buttons

Wrap a series of buttons with .btn in .btn-group.

HTML 
<div class="btn-group" role="group" aria-label="Basic example">
  <button type="button" class="btn btn-primary">Left</button>
  <button type="button" class="btn btn-primary">Middle</button>
  <button type="button" class="btn btn-primary">Right</button>
</div>

Button groups require an appropriate role attribute and explicit label to ensure assistive technologies like screen readers identify buttons as grouped and announce them. Use role="group" for button groups or role="toolbar" for button toolbars. Then use aria-label or aria-labelledby to label them.

These classes can also be added to groups of links, as an alternative to the .nav navigation components.

Active link Link Link

If you are using the .btn class on its own, remember to at least define some explicit :focus and/or :focus-visible styles.
